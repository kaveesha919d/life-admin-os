# Life Admin OS

Life Admin OS is a web app that helps users manage bills, subscriptions, renewals, and important deadlines in one place.

## Features
- Track bills and due dates
- Track subscriptions
- Manage renewal reminders
- Dashboard for upcoming urgent items

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- GitHub

Phase 1 setup complete. Core feature development coming next.
